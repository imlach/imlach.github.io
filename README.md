# Ross Imlach Portfolio - Modern Rebuild

This is the modernized portfolio site for Ross Imlach, Director of Photography.

## Tech Stack

- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **JavaScript**: TypeScript
- **Features**: Intersection Observer, lazy loading, responsive design

## Development

```bash
npm install
npm run dev
```

Dev server runs at http://localhost:3000

## Build

```bash
npm run build
npm run preview
```

## Structure

- `/index.html` - Landing page with FILM/LIVE split hero
- `/film.html` - Film portfolio (shorts, music videos)
- `/live.html` - Live events portfolio
- `/portfolio/` - Individual project pages (preserved from original)
- `/src/` - TypeScript and CSS source files
- `/images/` - Project images and thumbnails

## Key Features

- Split hero design separating Film and Live work
- Modern CSS Grid layout
- Smooth scroll animations with Intersection Observer
- Lazy loading images
- Mobile-first responsive design
- Dark minimalist aesthetic
- Preserved all original portfolio projects
